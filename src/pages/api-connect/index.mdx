IBM API Connect is an integrated API management offering, with capabilities and tooling for all phases of the API lifecycle. Key steps of the API lifecycle include create, secure, manage, socialize, and analyze. Learn more about IBM API connects offering and  capabilities from this audio presentation.
Below mentioned  lab exercises will walk you through designing, publishing, and securing APIs.
IBM API Connect has several out of the box personas. Some of these personas will be used in the context of these labs. The IBM API Connect V10.x Deployment WhitePaper has more detail.
