IBM API Connect is an integrated API management offering, with capabilities and tooling for all phases of the API lifecycle. Key steps of the API lifecycle include create, secure, manage, socialize, and analyze. Learn more about IBM API connects offering and  capabilities from the audio presentation.The [IBM API connect audio presentation ](https://community.ibm.com/) has more detail.
  
Below mentioned  lab exercises will walk you through designing, publishing, and securing APIs.
