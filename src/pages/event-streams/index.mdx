IBM Event Streams is based on the open-source Apache Kafka project. It Is a high-throughput, fault-tolerant, pub-sub technology that helps enterprises to implement Event-driven architecture(EDA). IBM Event streams uses wide range of connectors to connect the Enterprise core systems and retrieve large data streams to build responsive, engaging user experiences solutions.
